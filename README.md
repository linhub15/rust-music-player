# rust-music-player
A music player built in Rust (Rodio)

`cargo build` - restores all dependencies and ...builds

`cargo run` - runs in dev mode
* `p` - pause song
* `r` - resume song
* `s` - stop song

# Dependencies
rodio

## Dev dependencies
* libsdl2-dev needed for rodio
